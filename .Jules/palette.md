## 2024-05-24 - Navigation Tabs WAI-ARIA Attributes
**Learning:** For custom tab components built with vanilla JavaScript without a framework, WAI-ARIA attributes (`role="tablist"`, `role="tab"`, `role="tabpanel"`) and state attributes (`aria-selected`, `aria-controls`, `aria-labelledby`) are required. Additionally, decorative SVG icons inside buttons should have `aria-hidden="true"` to prevent redundant screen reader announcements. We also learned we must dynamically update the `aria-selected` attribute via JavaScript when the active tab changes.
**Action:** When implementing or modifying custom navigation tabs, ensure these WAI-ARIA states are present and dynamically updated along with CSS class changes. Always hide decorative SVG icons from screen readers using `aria-hidden="true"`.
